# ระบบจัดการการยืม–คืนหนังสือในห้องสมุด

โปรเจกต์ระบบจัดการการยืม–คืนหนังสือในห้องสมุด พัฒนาด้วย PHP และ MySQL เพื่อช่วยจัดการข้อมูลหนังสือ การยืม–คืน และการจองหนังสืออย่างเป็นระบบ ลดความผิดพลาดในการตรวจสอบสถานะหนังสือ และเพิ่มความสะดวกในการบริหารจัดการภายในห้องสมุด

## ความสามารถของระบบ

* จัดการข้อมูลหนังสือ (เพิ่ม แก้ไข ลบ และค้นหา)
* ระบบยืม–คืนหนังสือ
* ระบบจองหนังสือ
* ตรวจสอบสถานะหนังสือแบบเรียลไทม์
* บันทึกประวัติการยืม–คืน
* จัดการข้อมูลผู้ใช้งาน

## เทคโนโลยีที่ใช้

* PHP
* MySQL
* HTML
* CSS
* JavaScript

## วัตถุประสงค์

เพื่อพัฒนาระบบที่ช่วยลดขั้นตอนการจัดการงานภายในห้องสมุด เพิ่มความถูกต้องในการติดตามสถานะหนังสือ และช่วยให้การให้บริการผู้ใช้งานมีประสิทธิภาพมากยิ่งขึ้น
